Bienvenue sur le blog KoproGo
==============================

Nous inaugurons aujourd'hui le blog officiel du projet KoproGo !

Ce blog accompagnera le developpement de la plateforme open-source de gestion
de copropriete pour la Belgique. Vous y trouverez :

- **Annonces** : nouvelles fonctionnalites, mises a jour, jalons atteints
- **Articles techniques** : architecture hexagonale, choix de conception, retours d'experience
- **Communaute** : temoignages, contributions, evenements
- **Conformite** : GDPR, comptabilite belge (PCMN), loi sur la copropriete

Restez informes et suivez l'evolution du projet !
