   # Issue #026 - Modules Communautaires (Mission ASBL)
   
   **Priorité**: 🟢 NICE-TO-HAVE  
   **Estimation**: 15-20 heures  
   **Phase**: Phase 3 K8s Production  
   
   ## 📋 Description
   
   5 modules communautaires pour dynamique sociale et impact environnemental (mission ASBL).
   
   **Mission ASBL**: Résolution "phénomènes sociétés" via partage et solidarité
   
   ## 🎯 Modules
   
   1. **SEL (Système d'Échange Local)** - Troc compétences (monnaie locale virtuelle)
   2. **Swap Shop (Bazar de Troc)** - Échange/don objets entre habitants
   3. **Object Lending Library** - Prêt outils, échelles, tondeuse
   4. **Skills Directory** - Annuaire compétences (bricolage, cours)
   5. **Digital Notice Board** - Tableau affichage petites annonces
   
   ## 📐 Features
   
   - [ ] 5 entities: SelOffer, SwapItem, LendableObject, SkillOffer, Notice
   - [ ] Gamification: badges (Super Partageur, Éco-Héros)
   - [ ] Leaderboard copropriété
   - [ ] Modération: signalement contenu
   - [ ] Notifications: email + push nouveaux items
   - [ ] Tracking impact: CO2 économisé, objets réutilisés
   - [ ] Rapport annuel impact social (PDF pour AG)
   
   ## 📐 Impact Social
   
   - CO2 économisé via réutilisation
   - Objets détournés déchetterie
   - Compétences partagées (heures)
   - Économies réalisées (€)
   
   ## ✅ Critères d'Acceptation
   
   - 5 modules fonctionnels complets
   - Système gamification + badges
   - Outil modération
   - Dashboard impact social
   - Tests E2E chaque module
