Synthèse stratégique
--------------------

KoproGo se positionne comme **orchestrateur neutre d'achat groupé d'énergie**,
sans commission, entièrement transparent, conforme à la Charte CREG 2018
(décision B1614) et visant l'accréditation CREG (label qualité).

**Différenciateurs vs acteurs existants (iChoosr, Pricewise, Wikipower) :**

1. **Zéro commission** — Revenus KoproGo = abonnement SaaS uniquement
2. **Transparence totale** — Toutes les offres, méthodes de calcul publiées
3. **Calcul d'économies réel** — Basé sur le contrat actuel du consommateur, pas sur une moyenne marché
4. **Ouvert aux maisons individuelles** — Pas uniquement les copropriétés
5. **Communautés d'énergie** — Prêt pour CER/CEC (partage énergie locale)
6. **Label CREG ciblé** — Accréditation Charte B1614 (processus Jalon 3+)

**Base légale :**

- Loi du 29/04/1999 sur l'électricité
- Loi du 12/04/1965 sur le gaz
- Loi du 09/05/2019 (définition légale intermédiaire achat groupé)
- Charte CREG B1614 du 05/07/2018
- Directives UE RED II (2018/2001) + Marché intérieur électricité (2019/944)
- Décrets régionaux CER : Wallonie (05/05/2022), Flandre, Bruxelles
