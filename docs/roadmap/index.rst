========================
Roadmap 2025-2030
========================

Cette section présente la roadmap de KoproGo, organisée par capacités et jalons mesurables, pas par dates fixes.

.. note::
   **Philosophie** : Nous livrons quand les capacités sont atteintes, pas selon un calendrier arbitraire.

Vue d'Ensemble
==============

**Objectif** : 5 000 copropriétés belges utilisant KoproGo

**Progression par Paliers Mesurables**

* **Validation** (100 copros) : Beta publique
* **Viabilité** (500 copros) : Production ouverte
* **Impact** (1 000 copros) : Communauté active
* **Leadership** (2 000 copros) : Référence belge
* **Référence** (5 000 copros) : Leadership EU

**3 Phases Infrastructure**

1. **Phase 1 : VPS MVP** (2025-2026) - 100-500 copros
2. **Phase 2 : K3s + Automation** (2026) - 500-1 000 copros
3. **Phase 3 : K8s Production + PropTech 2.0** (2026-2030) - 1 000-5 000+ copros
