# Communauté d'énergie renouvelable : produire et partager l'électricité dans votre immeuble

Votre copropriété a un toit plat ou une façade ensoleillée inutilisée ?
Des panneaux solaires collectifs pourraient produire de l'électricité pour
tous les appartements, et même pour vos voisins dans des maisons individuelles.
C'est ce qu'on appelle une **Communauté d'Énergie Renouvelable (CER)**.

## Qu'est-ce qu'une Communauté d'Énergie Renouvelable ?

Une CER est une structure légale (ASBL, coopérative, ou autre) qui permet
à un groupe de consommateurs de **produire de l'énergie renouvelable ensemble**
et de partager cette production entre membres.

Ce n'est PAS un achat groupé classique. La différence fondamentale :

```
ACHAT GROUPÉ                    COMMUNAUTÉ D'ÉNERGIE (CER)
─────────────────────────────   ────────────────────────────────
Achète l'énergie sur le marché  Produit l'énergie localement
Fournisseur externe             Production propre (PV, éolien...)
Économie 8-15% sur la facture   Économie 30-60% sur la facture
Aucun investissement            Investissement initial requis
Durée : 1 an (renouvelable)     Durée : 20 ans (ROI installation)
Disponible partout              Requiert une installation de production
```

## Comment fonctionne le partage virtuel ?

Vous n'avez pas besoin de câbles directs entre les panneaux et les appartements.
Le partage se fait "virtuellement" via le réseau de distribution public :

1. Les **panneaux PV** sur la toiture de l'immeuble produisent de l'électricité
2. Cette énergie est injectée dans le réseau local (Fluvius/Ores/Sibelga)
3. Le gestionnaire de réseau (GRD) comptabilise la production vs la consommation
   de chaque membre
4. Chaque membre bénéficie d'une **déduction** sur sa facture de distribution
   proportionnelle à sa quote-part dans la production

```
[Panneaux PV 20 kWc sur toiture → 20,000 kWh/an produits]
         ↓ injection réseau
[Réseau Sibelga/Fluvius/Ores]
         ↓ déduction sur facture
[Appartement 1] : 1,500 kWh/an partagés (7.5%)
[Appartement 2] : 1,200 kWh/an partagés (6%)
[Maison voisine]: 800 kWh/an partagés (4%)
...etc.
```

## Combien peut-on économiser ?

Contrairement à l'achat groupé classique (où seule la composante "énergie"
est négociée), la CER permet d'économiser aussi sur les **frais de distribution**,
car l'énergie partagée localement est taxée à un taux réduit (tarif de réseau
simplifié pour l'énergie produite et consommée dans la même zone).

Exemple pour une copropriété de 20 appartements à Bruxelles :

```
Installation PV : 20 kWc (≈ 120 m²)
Production annuelle : ~18,000 kWh
──────────────────────────────────────────────────────
Par appartement (20 appareils, quote-part égale) :
- Énergie partagée   : 900 kWh/an
- Économie facture   : 150-200 €/an/appartement (30-40%)
──────────────────────────────────────────────────────
Coût installation    : 20,000-30,000 € (aide primes déduites)
ROI                  : 8-12 ans
Durée panneaux       : 25-30 ans
```

## Qui peut créer ou rejoindre une CER en Belgique ?

La Belgique a transposé la Directive européenne RED II (2018/2001) via des
décrets régionaux :

**Wallonie** : Décret du 5 mai 2022. Régulateur : CWaPE.

**Flandre** : Décret flamand d'électricité. Régulateur : Vlaamse Nutsregulator.

**Bruxelles** : Ordonnance électricité. Régulateur : BRUGEL.

**Qui peut être membre ?**

- Personnes physiques (copropriétaires, locataires, voisins en maison individuelle)
- PME locales
- Autorités locales (communes, CPAS, SLSP)
- Associations et coopératives

**Limite géographique** : Les membres doivent être dans la même zone de
distribution locale (en pratique : même quartier, même GRD, parfois rayon
de quelques km selon la région).

## Comment démarrer concrètement ?

**Étape 1 — Évaluation technique (AG copropriété)**

- Étude d'ensoleillement de la toiture (peut être faite gratuitement par
  les services énergie des communes ou des intercommunales)
- Estimation production kWh/an et coût installation
- Discussion AG : décision à la majorité des 4/5 (Art. 3.88 §2 CC — travaux
  importants sur parties communes)

**Étape 2 — Création de la structure légale**

- Créer une ASBL ou coopérative incluant les membres intéressés
- Rédiger les statuts (répartition production, règles d'adhésion/sortie,
  gestion des excédents)
- S'enregistrer auprès du régulateur régional (CWaPE/VREG/BRUGEL)

**Étape 3 — Installation et contrat GRD**

- Choix installateur (agréé, assurance décennale)
- Contrat de raccordement avec le GRD (Fluvius/Ores/Sibelga)
- Compteurs intelligents requis pour le partage virtuel (déjà déployés
  progressivement par les GRDs)

**Étape 4 — Démarrage et gestion**

- Activation du mécanisme de partage
- Facturation mensuelle aux membres (KoproGo peut automatiser)
- Rapport annuel à l'AG sur la production et les économies

## Les aides disponibles

**Wallonie** :

- Prime à l'investissement PV (Qualiwatt ou équivalent régional)
- Prêt à taux zéro (Écopack, MEBAR)
- Quota d'autoconsommation favorable

**Bruxelles** :

- Primes RENOLUTION (rénovation + énergie)
- Prime pour panneaux photovoltaïques (BRUGEL)

**Flandre** :

- Prime nette metering (controversée, en évolution)
- Aide à la consommation communautaire

**Niveau fédéral** :

- Déduction fiscale pour investissements économiseurs d'énergie
- TVA 6% sur les panneaux PV pour les bâtiments résidentiels (depuis 2023)

> **En pratique**
>
> **Checklist pour lancer une CER dans votre copropriété** :

- [ ] Vérifiez l'ensoleillement de votre toiture (outil PVGIS : re.jrc.ec.europa.eu/en/pvg)
- [ ] Demandez une étude gratuite à votre commune ou intercommunale énergie
- [ ] Inscrivez le point à l'OdJ de la prochaine AG (point 7 ou 8 selon sequence_odj)
- [ ] Obtenez le soutien d'au moins 4/5 des copropriétaires pour les travaux
- [ ] Contactez le facilitateur CER de votre région :
  - Wallonie : CWaPE — https://www.cwape.be/secteur/communautes-partage-energie
  - Flandre : REScoop (coopérative) — https://rescoop.eu
  - Bruxelles : BRUGEL — https://www.brugel.brussels
- [ ] Vérifiez les compteurs intelligents : Fluvius/Ores/Sibelga confirment

> **Base légale**
>
> - **RED II transposée** : Directive 2018/2001, Art. 22 (CER)
> - **Wallonie** : Décret du 5 mai 2022 modifiant le décret du 19 janvier 2017
> - **Travaux toiture (AG)** : Art. 3.88 §2 CC (majorité 4/5 pour travaux importants)
> - **Partage virtuel** : Ordonnance/Décret régional selon région
> - **Sources** : CWaPE (cwape.be), VREG (vreg.be), BRUGEL (brugel.brussels)
